To do list web app

*** Click on the '?' again, to toggle this info.

This web app is a static live demo, which means that the data you introduce in here is not stored in a db, therefore, whenever you reload this webpage, data will be lost.

This to do list web app allows you to create and manage tasks within lists wihin user profiles.


Features

  • Create, edit, login, logout or delete user profiles!
  • Choose an avatar from available designs, all avatars are designed by Freepik.
  • Create or delete task lists.
  • Create, check or delete tasks.
  • Responsive design.

Resources
